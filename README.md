# Taller_PatronObserve
Taller acerca del Patron Observe usando paradigma de Aspecto.

Como parte de los requerimientos tenemos una ventana con 3 botones y cada botón tiene un color, al hacer clic se cambiará el color de fondo de la ventana de acuerdo al color de dicho botón, los cambios de color que se realicen son guardados en el archivo historico.txt y mostrados por consola.
